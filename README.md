# Campus-warranty-system
这款校园保修微信小程序，为校园设施维护提供便利。师生发现设施损坏，无需四处奔波，在微信小程序里就能快速报修。填写所在区域、故障详情并上传照片，信息即刻传达至维修部门。维修人员通过小程序接收任务，安排维修并反馈进度，师生能随时查看。维修完成后，师生可评价服务。学校管理者还能统计分析报修数据，掌握校园设施状况。它让校园报修更高效，保障师生正常学习生活。
